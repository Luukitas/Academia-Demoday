Python

O que é Python

Python é uma linguagem dinâmica, interpretada, robusta, multiplataforma, multi-paradigma (orientação à objetos, funcional, refletiva e imperativa) e está preparada para rodar em JVM e .NET Framework. Lançada em 1991 por Guido van Rossum, é uma linguagem livre (até para projetos comerciais) e hoje pode-se programar para desktops, web e mobile.

Mão na Massa

Primeiramente, temos que instalar o Python para o sistema operacional, o mesmo pode ser baixado no link abaixo:

https://www.python.org/downloads/

A instalação é bem simples, protocolo NNF(Next-Next-Finish). Após instalarmos o Python, vamos instalar a IDE. Você pode trabalhar com sua IDE favorita para nossos exemplos. Nos exemplos usarei o Netbeans com plugin para Python.

Estarei levando em consideração que você também esteja usando o Netbeans. Após termos configurado o ambiente de trabalho iniciaremos o nosso projeto, para isso vá no menu Arquivo > Novo Projeto, selecione a opção Python, Python Project, clique em Próximo. Defina o nome do projeto e clique em Finalizar. Como é de praxe, após criar o projeto veremos um arquivo .py com o nome do projeto e já com o clássico "Hello World", ao executá será exibido no prompt abaixo a palavra "Hello World".

# To change this template, choose Tools | Templates # and open the template in the editor.
__author__ = "bruno"
__date__ = "$31/05/2010 06:59:46$"
if __name__ == "__main__":
print "Hello World"

As linhas 1 e 2 são cometários. A linha 4 e 5 são utilizadas para a documentação do projeto, onde são definidos o autor e a data do projeto respectivamente. Mais abaixo, na linha 8, tem uma condição onde verifica se a palavra reservada __name__ é igual à __main__. Isto quer dizer que o código abaixo somente será executado se este for o arquivo principal, ou seja, o arquivo que foi invocado pelo interpretador, seria como o método main() em Java. A linha 9 irá exibir na tela a mensagem "Hello World".

Vamos agora fazer uma modificação neste código. A linha 9 será alterado para o seguinte valor e em seguida iremos executar o programa:

print "Este exemplo é muito fácil!"

Como se pode verificar na figura anterior, ocorreu um erro na execução do programa, isto aconteceu porque a linha 9 possui caracteres acentuados. Isso não quer dizer que não é possível trabalhar com palavras acentuadas no Python, para resolver este problema é só inserir a seguinte linha na primeira linha do seu arquivo .py:

# -*- coding: iso-8859-15 -*-
